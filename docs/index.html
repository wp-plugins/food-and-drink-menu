Guide to the Food and Drink Menu plugin for WordPress

This guide will help you install and configure the plugin for your site.

Further support for this plugin can be found at themeofthecrop.com.

Installation

The following steps describe how to upload the Food and Drink Menu plugin to an existing WordPress installation. If you have not yet installed WordPress, consult the WordPress documentation for more information.

  1. Unpack the food-and-drink-menu.zip file. It should create a folder named /food-and-drink-menu/.
  2. Upload the /food-and-drink-menu/ folder to your WordPress plugin directory at /wp-content/plugins/.
  3. Go to your WordPress admin dashboard at yoursite.com/wp-admin and click the Plugins item in the menu on the left.
  4. Find Food and Drink Menu in the list of Plugins and click the Activate link below the plugin name.

The plugin is now active. Next you'll need to create your first menu.

Create a menu

To generate your menu, you'll create Menu Items and then add them to a Menu. Menu Items work just like posts. Go to the Menu Items > Add Menu Item page in the admin dashboard. Note: The Menu Items page will appear in the left menu, just like Posts, Pages and Comments, in your WordPress admin dashboard.

  1. Fill out the details for a Menu Item just like you would for a Page or Post. A Menu Item is one item from your menu, like a Butternut Squash Soup or Gourmet Burger.
  2. Look on the right, under the panel where you would find the Publish button, and find the Menu Sections area.
  3. You probably don't have any menu sections set up yet, so click the link to Add New Menu Section. Add the appropriate section for this item -- something like Starters, Desserts or Entrees.
  4. Go ahead and have a look around and you'll find other sections, including one where you can set a price. Once you're done, click the Publish button to publish this item.

Go ahead and create a couple more Menu Items. We'll want at least two Menu Sections before we create the menu, so make sure you've added a couple items to different sections.

When you're ready, go to the Menus > Add New Menu page in the admin dashboard and follow these steps to create your menu.

  1. Find the Menu Layout section. It should display the Menu Sections you set up earlier alongside two columns.
  2. Add a section to one of the columns by clicking on the section next to that column. Repeat the process until you've assigned all the sections to one or both of the columns.
  3. If you want to change the order of the sections or just remove one, click on it where it appears in the column. It will appear green where it has been "added" to the menu and grey when it is "waiting" to be added.
  4. Once you're happy with your menu, click on the Publish button. You'll then be able to view it by clicking the Preview Changes button, which appears above the Publish button.

You've got your menu set up now. But you probably want to link to it in your site menu or add it to an existing sidebar, page or post. don't leave this page just yet. Read the next section to find out how you can add the menu to any page, post or sidebar.

Link to the food and drink menu from your navigation menu

Your menus and menu items are now post types within WordPress, just like pages and posts. That means you can easily add them to any navigation menus you have set up on your site.

To do this, go to the Appearance > Menus page in the admin dashboard. If you're not sure how to use these menus, read the Easy WP Guide, which tells you all about how to use them.

You'll find all of your menus and menu items under the sections on the left, where you see sections like Pages, Links and Categories. If you don't see them there, look at the top right of your admin dashboard for the Screen Options tab. Click that to open the options and then make sure that Menus and Menu Items are checked.

Add the menu to a page, post or sidebar

You may want to add your menu or menu item directly to a page, post or sidebar. To do this, navigate to the menu or menu item in your admin dashboard and click the Edit button. This will open the page where you can edit your menu or menu item.

Look under the Publish panel on the right to find the Menu Shortcode panel (it will be in the column, but it might be below other panels as well). Here you'll find the shortcode you'll want to copy. It will look like: [fdm-menu id=1234]

You can paste this shortcode into the body of any page or post content and it will display your menu. You can find a shortcode for each Menu Item as well by looking at the same place on the editing page for those items. Learn more about you can customize the shortcode display.

The other way to display your menu is with a widget. Go to the Appearance > Widgets section in the admin dashboard. Find the Food and Drink Menu widget and the Food and Drink Menu Item widget and add either of them to any sidebar.

Advanced options

You can customize a few of the settings for this plugin by going to the settings page at Settings > Food and Drink Menu. Here you can disable specific features for the menu, such as the pricing information. This can be useful if you want to clean up the add/edit page details for a client.

You can also modify the thumbnail size that WordPress automatically generates for Menu Items. This will not necessarily change the size of the image as it is displayed on your website. You may need to change your theme's CSS code to change that. This setting only changes the size of the underlying image file that is served. You should only change these settings if you know what you're doing. Note: this does not work retroactively. Any photos already uploaded will not have the new sizes until you upload them again.

Shortcode parameters

This section is for advanced users who wish to use optional parameters with the shortcode.

At the moment, there are no custom options available for the menu shortcode. More will be added in the future.

Menu Shortcode

Prints a full menu with all sections.

[fdm-menu id=1 show_title=0 show_content=0]

  • id
    Accepts the id of the desired menu. Must be an integer.
  • show_title
    Whether or not to display the menu's title. Accepts 0 (hide title) or 1 (show title).
  • show_content
    Whether or not to display the menu's body content. Accepts 0 (hide content) or 1 (show content).

Menu Item Shortcode

Prints a single menu item.

[fdm-menu-item id=1]

  • id
    Accepts the id of the desired menu. Must be an integer.
